StreetMode - Mini static web app (black & white)
--------------------------------------------------
This is a single-page static site ready to deploy (Vercel, Netlify, any static host).

How to use:
1) Download and unzip the package.
2) Edit products.json to change images / prices / Yupoo links.
3) Replace assets (images) in /assets folder with real product images.
4) Upload the folder/zip to Vercel via 'Import Project > Upload' or push to GitHub and connect.
5) Orders: when user checks out, a mailto is created to 'streetmodeshop6@gmail.com' with order details. You then process on CSSBuy using the Yupoo links.

Email for orders: streetmodeshop6@gmail.com
Languages: Bosnian / English (toggle with top-right button)
